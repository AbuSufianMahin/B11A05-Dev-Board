Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in DOM :

  1. getElementByTagName()
  2. getElementById()
  3. getElementsByClassName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : it reads the whole HTML Code - both the tags of HTML language and text inside of it.

innerText : it only reads the textual content as it renders on screen inside of an element and ignores the tags.

textContent : it only reads the texual content without caring about styles. It can even read the hidden element's textual contends.

Question-3: What is event delegation in the DOM?

A technique in JavaScript where addEventListener() method is attached to a parent element instead of attaching it to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in JavaScript where an event triggered on a child element propagates upward through its ancestors nodes in the DOM. It allows parent elements to respond to the events that are triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element : document.createElement('tagName') can be used. This method takes an tag in its parameter.
To add an element : parentElement.appendChild(newElement) can be used.
To remove an element : parentElement.removeChild(newElement) can be used. It removes the child of parentElement assigned with the variable newElement. parentElement.remove() can be used to remove the element itself.
